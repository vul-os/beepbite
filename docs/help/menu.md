# Menu Management

The menu is the catalogue of items your customers can order. It is organised into categories, and items can have variations, modifiers and recipe costs.

---

## Categories

1. Go to **Menu → Categories**.
2. Tap **Add category** — give it a name and an optional display colour.
3. Categories appear as tabs on the POS and on the customer-facing menu.
4. Drag to reorder.

## Items

1. Inside a category, tap **Add item**.
2. Fill in: **name**, **price**, **description** (optional), **image** (optional).
3. Set `is_active` to make the item visible.
4. Assign the item to a **KDS station** if different from the category default.

## Variations

Variations let customers choose options (e.g. *Size: Small / Medium / Large*).

1. On an item, go to **Variations → Add variation group**.
2. Set a name (e.g. *Size*) and whether selection is required.
3. Add options with optional price adjustments (e.g. *Large +$1.00*).

## Modifiers

Modifiers are add-ons (e.g. *Extra cheese +$0.50*). Attach modifier groups to items in the same way as variations.

## Courses

Courses control KDS fire order (starter → main → dessert).

1. Go to **Menu → Courses**.
2. Add courses and assign a display order.
3. Tag items with their course. The POS will prompt staff to fire each course.

## Menu schedules

You can restrict items or categories to certain time windows (e.g. breakfast 07:00-11:00).

1. Go to **Menu → Schedules → Add schedule**.
2. Set days, start time and end time.
3. Assign the schedule to a category or item.

## Recipe costing

Attach ingredient recipes to items for food-cost tracking.

1. Go to **Inventory → Recipes → Assign recipe to item**.
2. The system calculates COGS from ingredient costs and portion sizes.

## AI menu creator

Use the **AI menu assistant** (Menu → AI import) to generate a starter menu from a description. Review and edit before publishing.
